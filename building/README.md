# Yacht Construction Guide

This directory contains the detailed step-by-step plan for building the yacht, from facility setup to final handover. The process is divided into chronological milestones.

## Directory Structure

### [00. Preparation & Procurement](./00_Preparation_Procurement/)
Getting ready to build.
- **Facility Setup**: Workshop, Jig, Strongback.
- **Material Procurement**: Resin, Glass, Core ordering.
- **Team Mobilization**: Hiring and Training.

### [01. Hull & Superstructure](./01_Hull_Superstructure/)
The composite shell construction.
- **Hull Lamination**: Layup of the main hull.
- **Structure & Grid**: Stringers, Bulkheads, Engine beds.
- **Deck & Superstructure**: Deck molding and bonding.
- **Glazing**: Installation of windows.

### [02. Systems & Propulsion](./02_Systems_Propulsion/)
Installation of major machinery and plumbing.
- **Propulsion**: Engine, Shaft, Propeller.
- **Plumbing**: Tanks, Bilge, Water systems.
- **HVAC**: Ventilation and Air Conditioning.

### [03. Electrical & Electronics](./03_Electrical_Electronics/)
Wiring and intelligence.
- **Cable Routing**: Pulling wires.
- **Panels**: Switchboards and Termination.
- **Electronics**: Navigation and Comm gear.

### [04. Interior & Fit-out](./04_Interior_Fitout/)
Making it livable.
- **Insulation**: Thermal and Acoustic.
- **Carpentry**: Furniture and Joinery.
- **Finish**: Paint, Varnish, Upholstery.

### [05. Commissioning & Trials](./05_Commissioning_Trials/)
Testing and Delivery.
- **Dock Trials**: Static testing.
- **Sea Trials**: Dynamic testing.
- **Handover**: Final delivery.

## How to use this guide
Each folder contains markdown files for specific steps. Each file details:
- **Requirements**: Pre-requisites.
- **What/Why/How**: Explanation of the task.
- **QA**: Quality Assurance checks.
- **Definition of Done**: Completion criteria.
- **Checklists & Cost estimates**.
